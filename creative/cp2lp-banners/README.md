# cp2lp diet banner test set

Landing page: `https://lp.zennuwellnessdesign.jp/cp2lp`

All four creatives keep the same photo, `熊本・上通アーケード内` local hook,
`3ヶ月平均 -6kg` proof, free-trial offer, color palette, and square format.
The main diet message changes between creatives.

| Creative ID | File | Main message | Hypothesis |
| --- | --- | --- | --- |
| DIET-01-RESULT | `diet-01-result-kamitori.png` | 食べて、落とす。 | A direct result-led promise creates the strongest response. |
| DIET-02-FOOD | `diet-02-food-kamitori.png` | 炭水化物を抜かずに、食べて痩せる。 | Freedom from strict carbohydrate restriction reduces resistance. |
| DIET-03-PERSONAL | `diet-03-personal-kamitori.png` | 自己流では変わらなかったあなたへ。 | Personalization resonates with people who have already failed alone. |
| DIET-04-LOOK | `diet-04-look-kamitori.png` | 体重だけでなく、鏡に映る見た目まで。 | Appearance change is more motivating than weight alone. |

Common offer: `食事改善＋パーソナル / 3ヶ月平均 -6kg / 初回体験 0円`

The `-6kg` figure is an existing-member average. It is not a guarantee of
individual results.
